\Hefteintrag{1.5}{Besondere Methoden: Main/Hauptprogramm}{%
Möchte man ein Java-Programm nicht mühsam mit einzelnen Objekte über BlueJ, sondern als normales Programm starten, benötigt man eine \emphColA{\LoesungLuecke{main-Methode}{8cm}}.

Diese nennt man auch den \emphColA{Einstiegspunkt/Hauptprogramm} eines Programms. In ihr werden meistens die benötigten \LoesungLuecke{Objekt}{5cm} erstellt und ihre \LoesungLuecke{Methoden}{5cm} aufgerufen.

Der \emphColC{Kopf} der Main-Methode ist immer gleich (siehe Bsp.):

\emphColC{public static void main(String[] args) \{}\\
~~~~~~\emph{Spieler s1 = new Spieler(100,50);}\\
~~~~~~\emph{// ... weitere Objekte und Methoden}\\
\emphColC{\}}

Das Schlüsselwort \LoesungLuecke{static}{3cm} bedeutet hierbei, dass kein Objekt benötigt wird, um die Methode aufzurufen. In BlueJ können solche Methoden mit Rechtsklick auf die Klasse aufgerufen werden.
}
